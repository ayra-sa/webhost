Malware Detection

Intrusion Detection and Protection System

Selain mengatasi serangan berbahaya di website, Imunify360 juga mampu mendeteksi potensi-potensi serangan. Imunify360 secara otomatis akan memblokir alamat IP yang terindikasi melakukan aktivitas mencurigakan di website Anda.

Proactive Defense

Proactive Defense mampu menghentikan malware yang bahkan tidak bisa dideteksi oleh scanner sekalipun! Fitur ini akan mengidentifikasi ancaman keamanan di website Anda secara real time dan memblokir potensi-potensi ancaman terhadap website Anda secara otomatis dalam waktu singkat.

Patch Management

Mengupdate kernel server terbaru adalah sebuah kewajiban untuk menjaga kinerja server Anda dalam performa terbaik. Patch Management memungkinkan kernel server Anda tetap terupdate ke versi terbaru tanpa perlu melakukan reboot dan juga tidak mengganggu kinerja server sama sekali.
